``` TestApp (SwiftUI) ```


Mi aplicación de notas es un proyecto que utiliza SwiftUI, el nuevo framework de Apple para crear interfaces de usuario de forma declarativa. La aplicación permite crear, editar y eliminar notas, así como sincronizarlas. He elegido SwiftUI porque me ofrece una sintaxis simple y natural de Swift, que me ayuda a mantener el código y el diseño perfectamente sincronizados. Además, con SwiftUI puedo crear una vista única que se adapta a diferentes dispositivos de Apple, aprovechando las ventajas de cada plataforma. Para gestionar la persistencia de datos, he utilizado CoreData, el framework de Apple que me permite modelar, almacenar y manipular los datos de mi aplicación de forma eficiente y sencilla. Con CoreData, he definido las entidades y las relaciones de mi modelo de datos, y he accedido a ellos mediante objetos. CoreData también me ha ofrecido ventajas como el agrupamiento, el filtrado, el ordenamiento y el faulting de los datos. Para asegurar la calidad del código, he implementado test unitarios y test de integración que comprueban el correcto funcionamiento de las clases, los métodos y las interacciones entre los componentes de la aplicación.
