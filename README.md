``` TestApp (SwiftUI) ```


TestApp es un proyecto que utiliza la arquitectura MVVM para organizar el código de forma limpia y reutilizable. 
La aplicación permite crear, editar y eliminar notas, así como sincronizarlas con una base de datos remota. Para asegurar la calidad del código, he implementado test unitarios y test de integración que comprueban el correcto funcionamiento de las clases, los métodos y las interacciones entre los componentes de la aplicación.
